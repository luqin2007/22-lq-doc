使用 `{}` 可以快速创建序列，生成的序列以空格分隔
- `{ a, b, c, d }`：枚举序列值
- `{ <start>..<end> }`：创建连续序列，可以是数字、字符等
- `{ <start>..<end>..<step> }`：序列包含步长

```bash
echo {0..9}
echo {0..9..2}
echo {a..z}
```

序列嵌套可以自动展开

```bash
echo { x, y{1,2,3}{a,b,c}, z }
```

> [!warning] 序列中不能使用变量展开
```bash
a=10
echo {0..10}
echo {0..$a}
```
